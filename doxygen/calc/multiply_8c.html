Parameters

[in]b第二オペランド。
[out]result計算結果を格納するポインタ。
Returns
成功時は CALC_SUCCESS、失敗時は CALC_SUCCESS 以外の値を返します。

この関数は 2 つの整数を受け取り、その積を result に格納します。

使用例
int result;
if (multiply(5, 4, &result) == CALC_SUCCESS) {
printf("Result: %d\n", result); // 出力: Result: 20
}
