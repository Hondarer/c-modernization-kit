2 クラス/構造体

2.1 CalcLib::CalcResult

計算演算の結果を表します。

2.1.1 プロパティ

2.1.1.1 IsSuccess

bool IsSuccess;

演算が成功したかどうかを示す値を取得します。

2.1.1.2 Value

int Value;

計算結果の値を取得します。 この値は IsSuccess が true の場合のみ有効です。

2.1.1.3 ErrorCode

int ErrorCode;

演算が失敗した場合のエラーコードを取得します。 0 (CALC_SUCCESS) は成功を、-1 (CALC_ERROR) は失敗を示します。
