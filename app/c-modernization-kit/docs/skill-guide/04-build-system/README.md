# ビルド システム (ステップ 3 - ビルド理解)

GNU Make・GCC / MSVC・.NET SDK を使ったビルドの仕組みを理解します。  
このリポジトリでは C プロジェクトと .NET プロジェクトが混在しており、それぞれのビルド手順を把握することが重要です。

## スキル ガイド一覧

| スキル ガイド                | 内容                                    |
|-----------------------------|-----------------------------------------|
| [GNU Make](gnu-make.md)                       | makefile の基礎と階層ビルド構造         |
| [GCC / MSVC ツール チェーン](gcc-toolchain.md) | コンパイラとリンカーのオプション          |
| [.NET SDK](dotnet-sdk.md)                     | dotnet コマンドと .NET プロジェクト構造 |

Table: スキル ガイド一覧

## このリポジトリとの関連

- `makefile` (トップレベル) - 全体ビルドのエントリ ポイント
- `framework/makefw/` - Make ビルド フレームワーク (サブモジュール)
- `app/calc/prod/` - C プロジェクトのビルド対象
- `app/calc.net/prod/` - .NET プロジェクトのビルド対象 (`dotnet build`)
