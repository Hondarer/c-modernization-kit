2 関数

2.1 subtract

int subtract (
    const int a,
    const int b,
    int * result
)

2 つの整数を減算します。

2.1.1 Parameters

  • [in] a 第一オペランド。
  • [in] b 第二オペランド。
  • [out] result 計算結果を格納するポインタ。

2.1.2 Return

成功時は CALC_SUCCESS、失敗時は CALC_SUCCESS 以外の値を返します。

2.1.3 Note

オーバーフローが発生する可能性がある場合は、 呼び出し側で範囲チェックを行ってください。

2.1.4 Warning

result が NULL の場合は失敗を返します。

2.1.5 使用例

int result;
if (subtract(10, 3, &result) == CALC_SUCCESS) {
    printf("Result: %d\n", result);  // 出力: Result: 7
}

2.1.8 Details

2.3 divide

int divide (
    const int a,
    const int b,
    int * result
)

2 つの整数を除算します。

2.3.1 Parameters

  • [in] a 被除数。
  • [in] b 除数。
  • [out] result 計算結果を格納するポインタ。

2.3.2 Return

成功時は CALC_SUCCESS、失敗時は CALC_SUCCESS 以外の値を返します。

2.3.3 Warning

ゼロ除算の場合、または result が NULL の場合は失敗を返します。

2.3.4 使用例

int result;
if (divide(20, 4, &result) == CALC_SUCCESS) {
    printf("Result: %d\n", result);  // 出力: Result: 5
}
