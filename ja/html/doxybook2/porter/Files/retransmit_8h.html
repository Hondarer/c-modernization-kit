2 関数

2.1 retransmit_init

void retransmit_init ( PotrRetransmit *rt, uint32_t timeout_ms, uint8_t max_retry )

再送制御管理構造体を初期化します。

2.1.1 引数

  • rt [out] 初期化する構造体へのポインタ。
  • timeout_ms [in] 再送タイムアウト (ミリ秒)。
  • max_retry [in] 最大再送回数。

2.2 retransmit_register

int retransmit_register ( PotrRetransmit *rt, uint32_t seq_num, uint32_t now_ms )

パケットを再送管理に登録します。

2.2.1 引数

  • rt [in,out] 再送制御管理構造体へのポインタ。
  • seq_num [in] 登録する通番。
  • now_ms [in] 現在時刻 (ミリ秒)。

2.2.2 戻り値

成功時は POTR_SUCCESS、エントリが満杯の場合は POTR_ERROR を返します。

2.3 retransmit_ack

int retransmit_ack ( PotrRetransmit *rt, uint32_t seq_num )

ACK 受信により再送管理から登録を解除します。

2.3.1 引数

  • rt [in,out] 再送制御管理構造体へのポインタ。
  • seq_num [in] 解除する通番。

2.3.2 戻り値

成功時は POTR_SUCCESS、エントリが見つからない場合は POTR_ERROR を返します。

2.4 retransmit_check

int retransmit_check ( PotrRetransmit *rt, uint32_t now_ms, uint32_t *seq_num_out )

タイムアウト発生パケットを検索して再送対象の通番を返します。

タイムアウトが発生したエントリを最も古い順に 1 件返します。
最大再送回数を超えたエントリは管理から削除します。

2.4.1 引数

  • rt [in,out] 再送制御管理構造体へのポインタ。
  • now_ms [in] 現在時刻 (ミリ秒)。
  • seq_num_out [out] 再送対象の通番を格納するポインタ。

2.4.2 戻り値

再送対象がある場合は POTR_SUCCESS、ない場合は POTR_ERROR を返します。

struct RetransmitEntry {
    uint32_t seq_num;
    uint32_t send_time_ms;
    uint8_t retry_count;
    uint8_t active;
    uint8_t[2] _pad;
}

再送制御エントリ。

送信済みパケット 1 つに対する再送管理情報です。

3.1.1.3 retry_count

uint8_t retry_count;
